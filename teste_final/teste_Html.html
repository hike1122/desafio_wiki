    Cores e seus tipos

    1o a padrao do vs code onde você pode botar o nome pronto. Ex (black, white...)

    2o RGB: é colocado valores de "0 á 255" em 3 fileiras. Ex (125,255,010)

    R: red, B: blue, G>green.

    3o HEX: é usado para escolher atraves de um codico usando o #. Ex (#ffffff -> branco)

    Hexadecimal. (HEX) um valor de 0-9 e A-F.

    4o HSL: para medir a cor, saturação e luz. Ex (0,0,0 -> branco)

    Hue, Saturation and Lightness. (HSL) H:0-360, S: 0-100%: L: 0-100%.

    Estruturas do HTML

    MAIN: Onde fica a informação principal do texto.

    HEADER: Cabeçario do HTML.

    FOOTER: Rodapé do HTML.

    FIGURE: Utilizado para botar figuras no Html que podem mudar dependendo do tamanho do dispositivo.

    NAV: Barra que serve de navegação do site.

    SECTION: Serve para botar uma area que não tem identificação.
